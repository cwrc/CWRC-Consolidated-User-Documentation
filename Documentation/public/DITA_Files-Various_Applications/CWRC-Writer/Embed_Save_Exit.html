Saving your document

In CWRC-Writer your documents can be edited by multiple people. The “Save and Exit” icon does as the title suggests and opens up the document for others to edit.

  1. Edit or Annotate your document.
  2. Once you have completed any changes you wish to make to the document click on the toolbar option to Save.
  3. Once you clicked on Save, the document validates and is successfully saved.
    If the document is not valid, a pop-up will appear and you will be asked if you would like to save anyway.

  4. If you would like to leave CWRC-Writer after saving your document, choose the Save and Exit option.
