Curating tags

The left column has a list of the tags found in the document. You can add new tags to the list, regardless of whether they have been included in your XML document or not.

  1. Click on the “Settings” menu in the top right corner.
  2. Select the “Curator” option under the “Mode” drop down menu. Click "OK."
  3. Select the “Add” button option at the bottom of the column.
  4. Enter in the XML tag in the right drop down menu and the label you would like to see accompany it. For example, in the “Tag/Path” column you would put ‘quote’ and in the left column put in ‘quote’ or ‘quotation.’
    For more information about adding different types of tags see the "Left Column of Dynamic Table of Context" reference document.
  5. Optional: If you would like to delete a tag or view where the tags are located in the “Table of Contents” or “Text Area.” Select one of other two options at the bottom of the page.
