        • Creating an XML document for the Dynamic Table of Contexts

          In the Dynamic Table of Contexts (DToC) users can upload their own XML text. These XML documents need to be specially formatted in order to experience the full range of capabilities provided by DToC.

        • Adding XML ids to your document manually

          DToC uses XML ids as index locators. Every index item is associated with one or more paragraphs; their function is analogous to that of page numbers used in print indexes.

        • Adding XML ids with oXygen

          This option is best for large DToC documents. The downside of using oXygen to insert IDs is that they are not human readable. It may be worth using the tool, if you are able to use a script to transform the ID pattern.

        • Adding an index to your XML document

          DToC is best configured for documents that have indexes.

Uploading your XML document to DToC

The information below is for an XML document that follows the template structure.

  1. Open up the settings in the homepage of DToC. This option is represented by a small grey circle in the top right corner of the “Add Texts” box.

  2. Change the “Input Format” to "DToC: Dynamic Table of Contexts."

  3. Enter your Title and Subtitle information into the corresponding areas of the "Corpus" dropdown menu.
  4. Click "OK".
  5. Click on the “Upload” button in the lower right corner.
  6. Select the XML document on your computer that you would like to upload to DToC.
  7. Voyant will process your document. Once it is finished processing your document, DToC will open with your XML document uploaded.
