Select by Set or Class

Select nodes by Set or Class using the Commands Tab.

  1. Click on the individual drawer of the desired Set or Class type to select contents.
  2. Click on a Verb to execute action.
    • Actions can be performed on multiple Set or Class types. Select as many drawers as needed before executing the action by clicking a Verb.
The drawer will turn a darker shade and the name of the Set or Class type will appear under the Current Command.

  • If a Class type was selected, the Edge types associated with that Class will appear as drawers in the Edges of Selected Nodes bin.


Set or Class types in a Selected state will remain so until either:
  • The drawer of the Set or Class type is clicked a second time to Deselect it.
  • The nodes are Discarded.
