Shelve

The Shelve function returns nodes to the the outer ring that orbits the central graph.

Shelving a node will place a nodes back on the Shelf, regardless of whether it is in a Graphed, Hidden or Discarded State.

Nodes can be shelved both manually and in the Commands Tab.
Note: To shelve ONLY Discarded nodes, use the Retrieve function.
