Viewing CWRC-Writer documents

CWRC users can view any uploaded XML documents in CWRC-Writer. If the document is locked by another user, you cannot edit or annotated the document until you get the other users premission.

  1. Open a tag or template in CWRC Writer.

    View is the default loading option when you open CWRC-Writer.







  2. Note: While in view mode CWRC-Writer does not allow you to change or add tags to the document.
