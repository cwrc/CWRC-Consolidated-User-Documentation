          CWRC users can view any uploaded XML documents in CWRC-Writer. If the document is locked by another user, you cannot edit or annotated the document until you get the other users premission.

General Description

CWRC-Writer provides two options for users . Users can viewor edit XML documents in CWRC-Writer.

When working with your document in the Edit mode other people cannot work with the document. If the document is locked (you donâ€™t have permission to work with it), someone is currently working with the document and you won't be able to go to the Edit Content view of the document.

If you are finished editing or annotating choose the Save and Exit option, this then frees up the document for other team members, so that they may work with it.

Viewing a document in CWRC-Writer

CWRC-Writer provides a viewing option so that users can look at locked documents (those that are not available to edit or annotate). This is the default view when users open CWRC-Writer.

While in view mode CWRC-Writer does not allow you to change or add tags to the document.
