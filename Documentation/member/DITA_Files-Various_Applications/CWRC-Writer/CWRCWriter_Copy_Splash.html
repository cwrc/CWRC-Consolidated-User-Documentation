        • Copying and pasting

          Copying and pasting is the easiest way to build the structure of your document.

        • Copying and pasting text

          Copying and pasting text into CWRC-Writer, can be achieved in two ways. The first, by highlighting text in the text area.

        • Copying and pasting a single tag

          Copying and pasting a tag allow you to duplicate structure and/or content with minimal effort. This acts as a type of shortcut for producing content in your document.

        • Copying and pasting multiple tags

          Copying and pasting multiple tags within CWRC-Writer is the fastest way to build up a document. Whether you are adding poem lines or more notes, you can create entire sections of structure in a few easy steps.

        • Viewing information copied from the structure panel

          CWRC-Writer provides the option of looking at what you have copied in the structure panel before you need to paste it in. This allows you to make sure you have selected exactly what you wanted to work with.

        • Copying and pasting entities in the editor

          You can copy and paste entities in CWRC-Writer. This will let you replicate annotations entities, so that you can have multiple instances of that tag in the document (with the same contextual information).
