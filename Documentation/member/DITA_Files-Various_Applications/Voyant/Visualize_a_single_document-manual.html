Visualize a single document (manual)

  1. Navigate to the document page within CWRC.
  2. Click on the Manage tab.
  3. Click on Create Bag(s).
  4. Copy the URL from the results page.
  5. Navigate to http://voyant-tools.org
  6. Paste the URL into the upload box and click on "Reveal".
