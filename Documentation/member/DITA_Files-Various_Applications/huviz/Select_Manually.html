Select Manually

Select nodes by hand with the following steps.

Note: All visible nodes, regardless of location (Shelf, Graph and Discard Bin), can be Selected manually.
  1. Hover over desired Node.
  2. Click node once to Select.
    The node will turn Bold, indicated that it has been Selected.

  • Click node a second time to Unselect. The node's appearance will change back to normal, indicating that is in a Unselected state.
