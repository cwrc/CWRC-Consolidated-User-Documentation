Execute Verbs in Commands Tab

In the Commands Tab, actions are automatically executed when a selected Set or Class type is combined with a Verb.

  1. Select nodes, either manually or by clicking on a Set or Class type.
  2. Click on a Verb to complete the formula and execute the action.
    Continue clicking on Verbs to execute a sequence of actions.
In this case, choosing the Set "Graphed" and then clicking the Verb "Label" resulted in the graphed nodes being labelled.



Note that the Verb in the Commands Tab is still blank. This is because, once a Set or Class type has been chosen, the user can continue to execute a series of Verbs.
