Searching CWRC

CWRC provides users with a search and advanced search functions. Searchable items include projects, collection, files and topics of interest.

  1. Enter your search term in the search box at the top of the page.
    You can narrow down where the search results come from, using the drop down menu. The options are: the project you are currently located in, the entire collaboratory or all website content.

  2. Click on the search icon.
  3. Optional: If you would like to narrow down your search results, you can use the Advanced Search feature.
    The Advanced Search function provides the opportunity to have multiple fields and search terms. Click on the red plus button to get more options. These are separated by Boolean operators (AND, OR, and NOT).

  4. Optional: Click on Search.
  5. Optional: Explore your search results.
    If you have multiple search results you can use the faceted browsing features along the left side of the results. These filters narrow down the search results by category. For more information about browsing CWRC see Searching and Browsing Reference.
