Introduction to Plot-It

Welcome to Plot-It. Here we describe how to get started using Plot-It and what to look out for.

To get started in Plot-It we suggest you familiarize yourself with the tool.

Plot-It is composed of a SIMILE Widgets timeline, a text box to view the event metadata, and a Google map. Users can browse and filter the events using the options on the left side of the pages. Users can also search for different events in the timeline and the map.
Note: If the entered events do not have place metadata they do not show up on the map. If there is no date attached to the event, it will not be available in the timeline.
Important: For the moment Plot-It is optimized only for FireFox and Chrome.
