Provide Your Own Dataset

You can load your own ontology and/or dataset to HuViz. External files need to be in Turtle (.ttl) format. There are two ways to upload your data: 1) local file or 2) URL. All files must be in ttl format.

  1. Click the Commands tab to open the main control panel.
  2. Click the dropdown menu labelled Dataset.
  3. Click Provide New Dataset.
  4. A new option will appear, asking you to "Drop File or URL."
    • Copy and past the file URL into the window.
    • OR open the file in your desktop finder. Drag and drop the file into the window.
    Once you drag the correct file type (.ttl) to the box, it will light up green

    .
  5. Click the dropdown menu labelled "Ontology."
    1. Choose a pre-loaded ontology from the dropdown menu.
    2. OR scroll down to "Provide New Ontology" to load your own ontology.
      Just like with a dataset you can upload your own ontology via local file or URL.
    Once you have chosen a dataset and an ontology, the "Load" button will appear.
  6. Click the "Load" button.
The dataset has sucessfully loaded and nodes are in orbit, ready for action.
