Show Edge Source

Clicking an edge displays the source of the data.

The Edge Inspector shows the source of the label names, the relational structure or ontology, as well as the file source.

  1. Hover over an edge linking two nodes together.

    The predicate label appears. The predicate describes the relationship between two nodes. In this example, "hasPoliticalAffiliation" indicates that Sophia King is affiliated with Radical politics.
  2. Click on the edge to open the Edge Inspector.

    The Edge Inspector pop-up window shows the source of the label names, ontology and file.
  3. Click on the URI for predicate to learn more about the relationship type and how it functions within the given ontology.
    The link takes the user to the CWRC ontology documentation, which describes the various properties of the predicate.
In this case, the tag "hasPoliticalAffliation" is associated with the subject Domain:"Person" and has various Children Properties or sub-categories, including: "hasGenderedPoliticalAffiliation" and "hasPoliticalAffiliationReported."
To close the window, click the button at the top right.
