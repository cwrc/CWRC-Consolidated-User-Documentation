Retrieve in Commands Tab

Use Commands to return Discarded nodes.

Move nodes from the Discard bin back to the Shelf using the Commands Tab.

  1. Under Sets, click Discarded.

  2. Under Verbs, click Retrieve.
The nodes in the Discard bin have returned to the Shelf

.
Note: To Retrieve only a specific Class of nodes (instead of all nodes):
  • Under Classes, click a Class Type
  • Under Verbs, click Retrieve




In this example, the Verb Retrieve was executed on the Class Type "Geography." As a result, the Geographic locations "England" and "Germany" returned to the Shelf. However, the Person nodes remained in the Discard Bin.
