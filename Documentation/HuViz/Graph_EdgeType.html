Graph by Edge Type

Graph nodes according to the type of connections (aka Edges) they share using the Edges of Selected Nodes bin.

  1. Select somes nodes. Remember, nodes can be Selected either by clicking on nodes in the graph or clicking on a Class in the Commands Tab.
    NOTE: When no nodes have been Selected, the Edges of the Selected Nodes bin will appear empty.

    Once you have Selected your nodes, the Edges of the Selected Nodes bin at the bottom of the Commands Tab will fill up with associated Edges. In this example, clicking on the class "relgion" to Select it has two effects. First, in the graph nodes classed as religions turn bold to show that they have been selected. Second, the Edges of the Selected Nodes bin shows two relationship types connected to "religion."



  2. Choose a type of connection by clicking on an Edge in the Edges of the Selected Nodes bin.
Nodes that share this Edge type will be dragged into the central Graph.

In this example, Selecting the class "Religion" shows the various religious organizations associated with the Virginia Woolf dataset. Clicking on the "hasReligion" predicate type in the Edges of the Selected Nodes bin further reveals the persons with religious affiliations. Viriginia Woolf, for instance, is affiliated with three religious groups.
