Open Saved Session

Open saved session.

Saved sessions can be opened from the HuViz main menu.
There are two types of saved sessions:
  • Saved to the HuViz menu
  • Saved as a file
To open a saved session:
  1. Click the Commands Tab.
    Note: If you have an existing session already open, the Commands Tab will not display the main menu. You need to restart the app (refresh the page or open a new window) to open a new session.
  2. Under "Script" click the Pick or Provide dropdown menu.
    • If the session is saved to the menu, select the session name from the dropdown menu.

    • OR if the session is saved as a file, select Provide New Script.

      Then, drag the URL or file into the highlighted box.



  3. Click Load to load file.


    NOTE: The saved session defaults to the first step. Click the forward and back arrows in the History Tab to see the latest changes.

The saved session is ready to use.
