How to Identify Selected Nodes

It is easy to tell the difference between Selected from Unselected nodes.

Table 1. How to Tell if a Node has been Selected:
Location Characteristic Image
Set/Class Bins
  • Selected Set types are brown
  • Selected Class types have a darker shade.

Current Command Line
  • If nodes were selected by type, the name of the Set or Class type will be listed.
  • If nodes were selected in the Graph (by manually clicking the nodes), an abreviated version of the node names will be listed.

Graph
  • Selected nodes will be bolded.
