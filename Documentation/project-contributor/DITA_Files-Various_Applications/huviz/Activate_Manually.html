Activate Manually

Graph nodes using the "drag and drop" method with the following steps.

  1. Click and hold the node, illuminating the circular green shelf.
  2. Drag and drop the node inside the circle.
Node has linked with a Hub and is now in an Active state.

Note: In the previous example, the node was dragged into the central Graph from the Shelf. However, a node can also be Activated by dragging it from the Discard bin.
