Sorting word frequencies

You can customize how you view word frequencies in DToC by different organizational headings and by how the words are displayed under those headings.

  1. Select the heading you would like to view the list of words organized by.
  2. Select whether you would prefer the list to be presented in ascending or descending order from the drop down menu.

  3. You an also select the “Column” option that displays another drop down menu filled with additional heading options which may be selected or deselected.
