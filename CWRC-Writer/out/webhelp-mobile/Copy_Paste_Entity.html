Copying and pasting entities in the editor

You can copy and paste entities in CWRC-Writer. This will let you replicate annotations entities, so that you can have multiple instances of that tag in the document (with the same contextual information).

  1. Right-click on the entity that you wish to copy, either in the entities column or within the document.

  2. Place your cursor in the text editor (wherever you would like the new entity to go) and right-click. Select the option to Paste Entity.
This will add a new entity in the document (you may have to delete texts as the new entity inserts beside any selected text).
