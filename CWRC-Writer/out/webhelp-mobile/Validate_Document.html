Validating your document

The validation of your XML document against the associated schema. If your document is invalid a list of errors found in the document will appear at the bottom of the page.

  1. Click on the Validate button on the toolbar..
    If the document is invalid a pop-up appears with error messages.

  2. Click on one of the errors, this will highlight the error in the document. You can then correct the invalid structure.
    You can still save the document if it is invalid, but a pop-up will appear making sure that you want to proceed.

  3. Validate again.
If you fixed all the validation errors, you should receive a confirmation message.
