Deleting tags in the editor

The document template that you are working with may have existing tags that you do not need for your document. You can easily delete these tags following these steps.

  1. Place your cursor in the text that is in the tag you want to delete. OR select the tag that you would like to delete in the structure panel.
  2. Right-click and choose the Remove Tag option.
This will delete the tag, but leave the text.
