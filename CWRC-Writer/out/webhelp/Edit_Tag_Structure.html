Editing tags in the structure panel

You can edit any of the tags that pre-exist in the template or that you have added to the document. This also allows you to add contextual information through attributes.

  1. Highlight on the tag that you would like to edit.
  2. Right-click on the tag and select the option Edit Tag.
  3. A pop-up will appear that allows you to add additional information to tags.
    The different attributes available in the pop-up will depend on the tag that you are editing.

Click OK and all edits made to the tag will be saved.
