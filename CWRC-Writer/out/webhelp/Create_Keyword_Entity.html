Creating a keyword entity

The "Tag Keyword" entity is tagging any important keywords in your document.

  1. Highlight the text that you would like to tag as a "Keyword." Click on the toolbar Keyword icon.
  2. Write the keyword in the text bar, and click Save.
This will create keywords in your document. You can edit those keywords in the entities column.
