Adding a bookmark

CWRC has a bookmarking feature so that you can access files across projects quickly. Additionally you can use your bookmark lists in the visualization tool PlotIt (for more information see Using PlotIt in CWRC )

You need to be logged in to your CWRC account to add bookmarks.
  1. Select the object you would like to bookmark.
    CWRC lets users bookmark subcollections (or folders) or individual files. What you bookmark depends on what you have open when you click on the Add to Bookmark button. For example, the screenshot below will add only that particular file to your bookmark list.

  2. Select the bookmark list you would like to add this bookmark to.
  3. Click on Add to Bookmark.
