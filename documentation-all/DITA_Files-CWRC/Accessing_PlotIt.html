Accessing PlotIt in CWRC

You can use the visualization tool, PlotIt, in CWRC. This feature works with your bookmark lists to map and date the files that you have bookmarked. For more information about using PlotIt, see

You need to be logged in to your CWRC account to use PlotIt.
  1. Open your account dashboard.
  2. Click on the Bookmark List tab on the right side of the page.
  3. Select the bookmark list that you would like to visualize in PlotIt.
  4. Click on View PlotIt - XMLDB.
  5. Explore PlotIt.
