General Description

CWRC-Writer provides three options for users to work with their documents. Users can view, annotate or edit documents.

When working with your document in the Annotation or Edit options other people cannot work with the document. If the document is locked (you don’t have permission to work with) or someone is currently working with the document you will be unable to “Annotate” or “Edit” and receive error messages.

If you are finished editing or annotating choose the “Save and Exit” option, this then frees up the document for other team members, so that they may work with it.

Viewing a document in CWRC-Writer

CWRC-Writer provides a viewing option so that users can look at locked documents (those that are not available to edit or annotate). This is the default view when users open CWRC-Writer.

While in view mode CWRC-Writer does not allow you to change or add tags to the document.



Annotating a document

This option allows users to work with the RDF in the document. You can add RDF annotations (person, place, organization, etc.) using the toolbar.

If the document is unavailable to be annotated or someone is currently working with the document you will be unable to “Annotate” your document and you will receive an error message.



Editing your document

This allows users to edit the content and tags of the document.
