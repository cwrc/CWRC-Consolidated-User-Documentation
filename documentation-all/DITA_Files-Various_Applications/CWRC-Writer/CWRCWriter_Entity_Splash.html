Tagging RDF entities

Tagging RDF annotated entities in CWRC-Writer allows users to add semantic information to certain text (person, place, etc).

Introduction to tagging entities

You can add annotated entities to Person, Place, Date, Organization, Event, Citation, Note, Text/Title, Correction, Keyword, and Link. You can add qualifying information to the entities like the attributes precision and identification.
