Copying and pasting text in the structure panel

This is how users can copy and paste text into CWRC-Writer by selecting a tag in the structure panel.

  1. Copy the information that you would like to paste into CWRC-Writer.
  2. Double click on the tag you wish to paste into in the structure panel until it is bolded and italicized.

    Note: If you use this method, you will delete all content currently in the tag that you are pasting into.
  3. With the tag you selected highlighted, use your keyboard shortcut to paste in the copied text.
