Visualize a Collection (manual)

CAUTION:
Note that at present all the documents to be visualized in Voyant need to come from a single collection. Sub-collections will not be visualized.
  1. Create or identify the collection that you want to visualize.
    Remember that you can select a large collection and then work with subsets of it within Voyant.
  2. Navigate to the collection within CWRC.
  3. Click on the Manage tab.
  4. Click on Bagit.
  5. Copy the URL from the results page.
  6. Navigate to http://voyant-tools.org
  7. Paste the URL into the upload box and click on "Reveal".
