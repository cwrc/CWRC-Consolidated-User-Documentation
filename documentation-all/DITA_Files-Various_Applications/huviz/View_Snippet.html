View Snippet Window

Snippet windows allow users to explore the relationships between nodes in more detail. In the HuViz prototype, the "snippet" of text is extracted from the source entry on the Orlando Project.

  1. Hover over the Edge linking two nodes together.

    A label will appear that categories the type of object and type of relationship. For example, "deathConnectiontoRegion" indicates that Ada Byron is connected to the region of Nottinghamshire by death.
  2. Click on the Edge to open the Snippet window.

To close window, click the button at the top right.
