Pin

The Pin function fixes or "pins" Graphed nodes in place.

By default, networks of graphed nodes float freely within the central stage. However, there are times when working with a static graph is preferable.

Using the Pin function, the branches of a node network can be stretched out and fixed in place.

Note: Only graphed nodes can be pinned. The Pin function cannot be executed on nodes that are in a shelved or discarded state.
Note: Graphed nodes can be Pinned in any area that is not designated as part of the Shelf or Discard Bin. This includes the area both inside and outside the central ring.
