Using the Graph

The quickest and most intuitive way to manipulate the dataset is directly in the graph.

Almost all actions can be performed on the nodes either through a single click or a "drag and drop."

Note: The exception is Label and Hide, which can only be performed in the Commands Tab.
Navigating the Graph:
  1. central Graph
  2. Shelf
  3. Discard Bin


Note: Nodes can be dragged between any of the three main areas: central Graph, Shelf and Discard Bin.
