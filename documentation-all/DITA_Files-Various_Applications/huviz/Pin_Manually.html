Pin Manually

Pin nodes using the "drag and drop" method with the following steps.

  1. "Click and hold" to grab a graphed node.
  2. Without releasing the mouse, drag the node to an unmarked area.
  3. Release the mouse to "drop" the node and fix in place.
Click and drag the node a second time to Unpin (release) the node.
