Change view mode

The user can change the way that they view the events from the timeline. The available options are a map, a list, and a grid view.

Between the map and the timeline there is the option to change the view of the events from the timeline (see image below). The options are “Map View”, “List View,” and “Grid View.” Select the option you would like.

In the List and Grid views, the user can change the way that the information is ordered. For example, if you would like to view events ordered from earliest to latest, click on the "Date" heading in the right column.
