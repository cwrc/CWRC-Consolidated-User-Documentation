Creating Index Subheadings

You can organize a group of indexes together under a subheading. You can add another list tag within the established list (anywhere in the list is fine).

  1. Nest a ‘list’ tag within an ‘item’ tag.
    An example index:

  2. Within the 'list' tag add an 'item' tag. So the breadcrumb trail would be item>lists>item(s). Each 'item' tag wraps around each index entry (it’s content is the index term).
  3. Step 3: Add a <ref> tag to the items in the list.

    The 'ref' tag has a target attribute, its value is a space seperated list of paragraph XML IDs. The 'ref' tag references the paragraphs in the body text.
  4. Begin to add your index terms between the item/ref tags.
  5. Now note all the paragraphs that your terms are found in. It may be helpful to write this down before you begin creating your index, or you can add the XML ids to your index as you go.
  6. Add the XML ids (from the paragraph(s) or other tags) in the target attribute of the <ref> tag. If there are multiple XML ids related to the index term, separate these with a space.
