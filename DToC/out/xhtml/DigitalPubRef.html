Digital Publication Options

DToC allows users to digitally “publish” their DToC documents by providing them with an URL or by opening a new page with the XML structure displayed with the document.

Export Document

If you would like to share your document with others or bookmark the URL without uploading the document again then you can choose to “Export” your document. DToC provides you with an URL to preserve a link of your curated document. The “Export” option is found under the settings menu option (upper right corner).

The Export option is used to create and preserve a link to the curated version of a document/edition.

Once you have opened the settings menu choose the “Export” button. This opens another popup that contains the URL for your document. You can either click on the linked “URL” option, which opens a new browser window or copy and paste the URL.


XML Digital View

If you would like to see your XML document structure (for XML documents only) you can choose the XML button, located beside the settings button in the upper right corner. This will open a new browser window with a display of your document XML structure.


You can bookmark this page. If you try to print this version of your document, the XML structure will not be printed, just the text.
