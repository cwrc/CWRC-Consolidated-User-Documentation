Uploading Text and URLs into DToC

If you do not have an XML document document, DToC allows you to upload text or URLs. Some features will be unavailable using this method (like the index).

  1. Paste your text or an URL into the text box located on the front page of DToC.

  2. Click on the “Reveal” button located right below the text box. This opens the document in DToC.
  3. Begin to work with your text.
    Using this method, you will not have an index or any tags (even if you paste in an XML document, see screenshot below). Additionally, you will not have a table of contents or lines in the document model and you cannot search the document. However, you will be able to see information in the stats column (term frequency).
