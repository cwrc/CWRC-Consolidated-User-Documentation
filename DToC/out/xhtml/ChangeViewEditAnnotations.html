Changing who can view/edit annotations

You need to login to use any annotation features

  1. Highlight any text you would like to annotate.
  2. Select one of the View/Edit options (see image below).
    You can check both options, so that they have the option either view or edit the annotations.

  3. Add any comments or tags to the highlighted text.
  4. Click on “Save.”
