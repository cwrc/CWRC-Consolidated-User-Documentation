Curating Tags

A list of tags found in the document is located in the left column. You can add new tags to the list, those that are found in your XML document or not.

  1. Click on the “Settings” menu in the top right corner.
  2. Select the “Curator” option under the “Mode” drop down menu.
  3. Click “Ok”.
    This changes the organization of the tags. They are ordered into two columns (“Tag/Path” and “Label”.

  4. Select the “Add” button option at the bottom of the column.
  5. If you are adding a tag that already exists in your document but is not in the list of tags it will be available in the drop down menu (see below).
  6. Enter in the XML tag in the right drop down menu and the label you would like to see accompany it. For example, in the “Tag/Path” column you would put ‘quote’ and in the left column put in ‘quote’ or ‘quotation.’
  7. Optional: If you would like to add tags that are not found in XML you can add them as targets. For example, if you would like to add a religion tag you can add it as a target of the ‘ref’ tag. It would look like this: ref{@target*='religion']
    When you switch back to the “Default” mode that tags you have added will be added to the original list.

  8. Optional: If you would like to delete a tag or view where the tags are located in the “Table of Contents” or “Text Area.” Select one of other two options at the bottom of the page.
